# 强直性脊柱炎运动康复治疗方案研究计划

## 研究目标
深入研究强直性脊柱炎患者的运动康复治疗方案，涵盖8个核心方面，并收集专业康复医学资料和临床实践指南。

## 研究范围
1. 物理治疗的重要性
2. 脊柱伸展运动
3. 呼吸训练
4. 姿势矫正练习
5. 关节活动度训练
6. 有氧运动建议
7. 日常生活活动指导
8. 运动禁忌症和注意事项
