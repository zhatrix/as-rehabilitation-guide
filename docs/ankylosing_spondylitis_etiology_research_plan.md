# 强直性脊柱炎病因学和发病机制研究计划

## 研究目标
深入研究强直性脊柱炎（Ankylosing Spondylitis）的病因学和发病机制，涵盖以下五个核心方面：
1. 遗传因素（HLA-B27等）
2. 环境触发因素 
3. 免疫系统异常
4. 炎症反应机制
5. 疾病进展过程

## 研究策略
- 优先使用权威医学数据库和官方医学机构资料
- 收集最新的研究文献和临床指南
- 确保信息的科学性和准确性
- 整合多个独立来源的验证信息
