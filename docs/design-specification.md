# 设计规范 - 强直性脊柱炎康复指导网站

**版本**: 1.0 | **风格**: 现代科研风格（基于瑞士设计原则） | **创建时间**: 2025-11-03

---

## 1. 设计方向与理由

### 1.1 风格定位

**现代科研风格**：以瑞士设计（国际排版风格）为基础，结合医学科研领域特征，采用等宽字体系统营造精准、技术、严谨的科研氛围。此风格强调网格化布局、层级分明的内容组织、极简的视觉装饰，使24,000字的医学文献内容获得最佳可读性与权威感。

**核心特征**：
- 95%中性色（黑白灰）+ 5%医学蓝色强调
- 等宽字体全局应用（科研报告质感）
- 严格8px网格系统
- 数学化的12列栅格布局
- 极简交互（快速响应、功能导向）

### 1.2 实例参考

- **PubMed/NCBI网站**：医学文献数据库，等宽字体用于数据精确呈现
- **Nature期刊数字平台**：科研论文排版，清晰的层级与表格设计
- **WHO指南网站**：医学权威性，严谨的栅格与中性色调

### 1.3 适用性分析

**为何选择此风格**：
- **内容为王**：24,000字医学文献需要极致可读性，瑞士设计的清晰层级与网格系统最适合
- **权威信任**：医学健康网站需要专业感，中性色+等宽字体传递科研严谨性
- **数据密集**：30个表格需要规整的表格设计，等宽字体确保数据列对齐
- **长时阅读**：用户需要深度学习，极简设计减少视觉疲劳

**权衡取舍**：
- ✅ 获得：最大化可读性、永恒的专业感、快速加载、清晰的内容层级
- ❌ 牺牲：品牌个性（极度中性）、视觉吸引力（无装饰）、情感连接（理性导向）

## 2. 设计Token系统

### 2.1 色彩系统

**色彩分布原则**：95%中性色（结构与内容） + 5%医学蓝（强调与交互）

#### 主色系 - 医学蓝（Medical Blue）

| Token | 值 | 用途 |
|-------|-----|------|
| primary-50 | `#E3F2FD` | 轻微背景、hover浅色 |
| primary-500 | `#0277BD` | 主要CTA、链接、医学图标 |
| primary-700 | `#01579B` | CTA hover、活动状态 |
| primary-900 | `#01395E` | 深色强调 |

**WCAG验证**：
- primary-500 (#0277BD) on white: 5.84:1 ✅ AA级
- primary-700 (#01579B) on white: 8.59:1 ✅ AAA级

#### 中性色系 - 灰阶

| Token | 值 | 用途 | WCAG |
|-------|-----|------|------|
| neutral-900 | `#000000` | 主标题、强调边框 | 21:1 ✅ AAA |
| neutral-800 | `#1A1A1A` | 二级标题 | 16.9:1 ✅ AAA |
| neutral-700 | `#333333` | 正文文本 | 12.6:1 ✅ AAA |
| neutral-500 | `#666666` | 次要文本、说明 | 5.74:1 ✅ AA |
| neutral-400 | `#999999` | 禁用状态、辅助信息 | 2.85:1 |
| neutral-300 | `#CCCCCC` | 边框、分割线 | - |
| neutral-100 | `#F5F5F5` | 轻微背景、表格斑马纹 | - |
| neutral-0 | `#FFFFFF` | 主背景 | - |

#### 语义色（仅功能性使用）

| Token | 值 | 用途 |
|-------|-----|------|
| success-600 | `#2E7D32` | 成功提示 |
| warning-600 | `#ED6C02` | 警告信息 |
| error-600 | `#D32F2F` | 错误/禁忌提示 |

#### 背景层次

| Token | 值 | 用途 |
|-------|-----|------|
| bg-base | `#FFFFFF` | 页面主背景 |
| bg-surface | `#F5F5F5` | 卡片、表格表头 |

### 2.2 字体系统

**特殊要求说明**：全局使用等宽字体（monospace），打破瑞士设计传统的Helvetica规范，以营造科研报告的精准质感。

#### 字体族

| Token | 值 | 用途 |
|-------|-----|------|
| font-primary | `'Roboto Mono', 'Consolas', 'Monaco', 'Courier New', monospace` | 全局字体 |
| font-code | `'Fira Code', 'Roboto Mono', monospace` | 代码片段（若有） |

#### 字号体系（桌面端，1920px基准）

| Token | 大小 | 字重 | 行高 | 用途 |
|-------|------|------|------|------|
| text-display | `56px` | 700 (Bold) | 1.1 | 页面主标题（h1） |
| text-h2 | `40px` | 700 (Bold) | 1.2 | 章节标题（h2） |
| text-h3 | `28px` | 600 (Semibold) | 1.3 | 小节标题（h3） |
| text-h4 | `20px` | 600 (Semibold) | 1.4 | 表格标题、卡片标题（h4） |
| text-body-lg | `18px` | 400 (Regular) | 1.6 | 引言、重点段落 |
| text-body | `16px` | 400 (Regular) | 1.5 | 正文（标准） |
| text-small | `14px` | 400 (Regular) | 1.5 | 图注、表格数据 |
| text-caption | `12px` | 400 (Regular) | 1.4 | 元数据、脚注 |

**等宽字体特性**：
- 字符宽度一致，适合数据对齐
- 科技感、精准感强
- 长文阅读需增加行高（1.5-1.6）以补偿可读性

#### 响应式字号（移动端 <768px）

| Token | 移动端大小 |
|-------|-----------|
| text-display | `36px` |
| text-h2 | `28px` |
| text-h3 | `20px` |
| text-body | `16px` (不缩小，保持可读性) |

### 2.3 间距系统

**8px网格**（严格遵守）

| Token | 值 | 用途 |
|-------|-----|------|
| space-xs | `8px` | 内联元素间距（图标+文字） |
| space-sm | `16px` | 相关元素间距 |
| space-md | `24px` | 段落间距、列表间距 |
| space-lg | `32px` | 章节内部padding |
| space-xl | `48px` | 主要模块间距 |
| space-2xl | `64px` | 页面区域分隔 |
| space-3xl | `96px` | 大型区域间距 |

### 2.4 圆角系统

| Token | 值 | 用途 |
|-------|-----|------|
| radius-none | `0px` | 默认（保持矩形锐利感） |
| radius-sm | `2px` | 按钮、输入框（微妙圆角） |
| radius-md | `4px` | 卡片、表格（若需柔和感） |

**原则**：默认0px（瑞士设计精神），仅按钮/输入框可用2px。

### 2.5 阴影系统

**最小化使用**（瑞士设计扁平化原则）

| Token | 值 | 用途 |
|-------|-----|------|
| shadow-sm | `0 1px 3px rgba(0, 0, 0, 0.12)` | 卡片（若需层次） |
| shadow-md | `0 2px 6px rgba(0, 0, 0, 0.16)` | Modal弹窗 |

**规则**：仅在功能必需时使用（如弹窗需要脱离层），避免装饰性阴影。

### 2.6 动画系统

| Token | 值 | 用途 |
|-------|-----|------|
| duration-fast | `150ms` | 按钮hover、焦点状态 |
| duration-base | `200ms` | 页面元素过渡 |
| easing-linear | `linear` | 功能性动画（优先） |
| easing-out | `ease-out` | 内容淡入 |

**原则**：极简、快速、功能导向。禁止>300ms的装饰性动画。

## 3. 核心组件规范

### 3.1 按钮 (Button)

**Primary按钮（主要操作）**：
- 高度：48px（移动端44px以上，满足触摸目标）
- 横向padding：24px
- 圆角：2px
- 字体：text-small (14px)，Bold 700，字母间距 0.05em
- 背景色：primary-500
- 文字色：白色
- Hover：背景色变为primary-700
- 禁用：背景色neutral-400，文字色neutral-500

**Secondary按钮（次要操作）**：
- 尺寸同Primary
- 背景色：透明
- 边框：2px solid neutral-900
- 文字色：neutral-900
- Hover：背景色neutral-900，文字色白色（反转）

**应用场景**：
- Primary：提交表单、跳转核心页面
- Secondary：取消、返回、辅助操作

### 3.2 表格 (Table)

**结构**：
- 表头背景：bg-surface (#F5F5F5)
- 表头文字：text-small (14px)，Semibold 600，全大写
- 单元格padding：12px 16px
- 边框：1px solid neutral-300
- 斑马纹（可选）：奇数行bg-base，偶数行neutral-100
- 数据文字：text-small (14px)，Regular 400

**数据对齐**（等宽字体优势）**：
- 文本列：左对齐
- 数字列：右对齐（等宽字体天然对齐小数点）
- 混合列：左对齐

**响应式策略**：
- 桌面：标准表格
- 移动：水平滚动容器包裹，或转为卡片堆叠

### 3.3 卡片 (Card)

**标准卡片**：
- 背景：bg-base (白色)
- 边框：1px solid neutral-300（或无边框）
- 圆角：0px（保持矩形）
- 内padding：32px（桌面），24px（移动）
- 阴影：可选 shadow-sm（若需层次感）
- Hover：边框变为primary-500（若为可点击卡片）

**卡片内部结构**：
- 标题：text-h4 (20px)，Semibold 600
- 正文：text-body (16px)，间距space-md (24px)
- 图片：宽度100%，纵横比保持

### 3.4 导航栏 (Navigation)

**桌面导航**：
- 高度：64px
- 背景：bg-base (白色)
- 底部边框：1px solid neutral-300
- Logo：左对齐，高度24px
- 导航链接：text-small (14px)，Bold 700，全大写，字母间距0.05em
- 链接间距：space-xl (48px)
- 活动状态：文字色primary-500，底部2px边框

**移动导航**：
- 汉堡菜单（右上角）
- 抽屉式侧边栏（从右滑入）
- 菜单项垂直排列，高度48px，padding 16px

### 3.5 媒体组件

#### 图片查看器 (Image Viewer)

**缩略图状态**：
- 宽度：100%（适应容器）
- 纵横比：保持原始比例
- 边框：1px solid neutral-300
- Hover：边框变为primary-500，鼠标指针cursor:zoom-in

**放大状态（点击后）**：
- 全屏Modal遮罩（背景rgba(0,0,0,0.9)）
- 图片居中，最大宽度90vw，最大高度90vh
- 关闭按钮：右上角，白色×图标
- 点击遮罩/按钮/ESC键关闭

#### 视频播放器 (Video Player)

**播放器外观**：
- 宽度：100%
- 纵横比：16:9（标准视频）
- 边框：1px solid neutral-300
- 控制条背景：半透明黑色rgba(0,0,0,0.7)

**控制功能**：
- 播放/暂停按钮（空格键快捷）
- 进度条（可拖拽）
- 时间显示（当前/总时长）
- 音量控制
- 全屏按钮
- 倍速控制（0.5x, 1x, 1.5x, 2x）

**状态反馈**：
- 播放中：隐藏控制条3秒
- 暂停：显示控制条
- 加载中：居中spinner

### 3.6 数据展示卡片 (Data Card)

**用于首页快速统计**：
- 尺寸：正方形或4:3比例
- 背景：bg-surface (#F5F5F5)
- 内padding：24px
- 数字：text-display (56px)，Bold 700，primary-500
- 标签：text-small (14px)，neutral-700
- 布局：数字居中，标签在下

## 4. 布局与响应式设计

### 4.1 栅格系统

**12列栅格**（基于瑞士设计原则）：
- 容器最大宽度：1200px（桌面）
- 列间距（gutter）：24px
- 外边距：最小48px（桌面），16px（移动）

### 4.2 页面布局模式

**基于content-structure-plan.md的4个页面**：

#### 首页布局 (index.html)

**区域流程**（由上到下）：
1. **Hero区** (高度400-500px)
   - 全宽容器
   - 内容区居中6列（50%宽度）
   - 标题 + 简介 + CTA按钮

2. **三大板块导航区** (space-2xl间距)
   - 3列卡片网格（桌面4-4-4列，移动堆叠）
   - 卡片高度auto，内容最小150px

3. **快速统计区**
   - 4列数据卡片（桌面3-3-3-3列，平板2-2排列，移动堆叠）
   - 间距space-lg (32px)

4. **疾病特征区**
   - 7-5列不对称分割（左文字，右图片）
   - 移动端垂直堆叠

#### 内容页布局 (etiology.html, medication.html, rehabilitation.html)

**标准模式**：
1. **页面标题区** (高度200px)
   - 8列居中
   - 标题 + 引言

2. **主内容区**
   - 8列居中（66.7%宽度，最佳阅读行长50-70字符）
   - 文本块、表格、图片遵循此列宽
   - 视频可扩展至10列（83.3%宽度）

3. **侧边栏（可选）**
   - 8列主内容 + 4列侧边栏（目录/相关链接）
   - 移动端侧边栏置于内容下方

### 4.3 响应式断点

| 断点 | 宽度 | 栅格 | 说明 |
|------|------|------|------|
| sm | 640px | 4列 | 手机横屏 |
| md | 768px | 8列 | 平板竖屏 |
| lg | 1024px | 12列 | 桌面 |
| xl | 1280px | 12列 | 大屏 |

### 4.4 响应式适配策略

**<768px (移动)：**
- 多列网格→垂直堆叠
- 字号缩小（见2.2节）
- 导航→汉堡菜单
- 表格→水平滚动或卡片化
- Hero高度减至300px

**768-1024px (平板)：**
- 8列栅格
- 保持部分两列布局
- 字号介于移动与桌面之间

**>1024px (桌面)：**
- 完整12列栅格
- 标准字号
- 水平导航

### 4.5 触摸目标

**移动端交互标准**：
- 所有可点击元素最小44×44px（Apple HIG）
- 推荐48×48px（Material Design）
- 元素间距最小8px

## 5. 交互与动画规范

### 5.1 页面加载

**初次加载**：
- 淡入动画（opacity 0→1，duration-base 200ms）
- 内容从上到下依次出现（延迟50ms递增）

**页面切换**：
- 淡出当前页（150ms）
- 淡入新页面（200ms）
- 滚动位置重置到顶部

### 5.2 按钮交互

**状态变化**：
- Hover：背景色/边框色过渡，duration-fast (150ms)
- Active（点击）：轻微缩放scale(0.98)
- Focus：2px蓝色轮廓（primary-500），无glow

**视觉反馈**：
- 点击立即响应（<150ms）
- 提交后显示loading spinner

### 5.3 图片放大交互

**触发方式**：
- 桌面：hover显示放大图标，点击放大
- 移动：直接点击放大

**动画**：
- 遮罩淡入（200ms）
- 图片从缩略图位置缩放至中心（300ms，ease-out）

**关闭方式**：
- 点击遮罩
- 点击关闭按钮
- 按ESC键
- 关闭动画反向播放（200ms）

### 5.4 视频播放交互

**自动行为**：
- 默认不自动播放
- 滚动到视口内不触发播放（用户主动控制）

**播放控制**：
- 点击播放按钮/视频画面：播放/暂停切换
- 空格键：播放/暂停
- 方向键←→：快进/后退5秒
- 方向键↑↓：音量增减10%

**进度显示**：
- 鼠标悬停进度条：显示时间tooltip
- 拖拽进度条：实时预览帧（可选）

### 5.5 表格交互

**桌面端**：
- Hover行：背景色变为neutral-100
- 可排序表头：显示排序图标，点击切换升序/降序

**移动端**：
- 水平滚动：显示滚动阴影提示
- 卡片模式：每行数据转为独立卡片

### 5.6 导航交互

**桌面导航**：
- 链接Hover：文字色变为primary-500，下划线从左滑入（width 0→100%，200ms）
- 活动页面：下划线常驻，文字色primary-500

**移动导航**：
- 汉堡菜单点击：侧边栏从右滑入（300ms，ease-out）
- 遮罩淡入（200ms）
- 点击遮罩/关闭按钮：滑出（200ms）

### 5.7 滚动行为

**平滑滚动**：
- 锚点跳转：smooth scroll，800ms
- 回到顶部按钮：页面滚动>200vh时淡入

**滚动性能**：
- 仅使用transform和opacity动画（GPU加速）
- 禁止在滚动时触发复杂计算

### 5.8 无障碍支持

**键盘导航**：
- Tab键：按逻辑顺序遍历所有交互元素
- Enter/Space：激活按钮/链接
- ESC：关闭Modal/弹窗

**屏幕阅读器**：
- 图片添加alt描述
- 按钮添加aria-label
- 表格添加scope属性
- 视频添加字幕轨道（若有）

**减少动画模式**：
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
